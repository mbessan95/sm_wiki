		Éditez les fichiers dans `src/content/docs/` (Markdown ou MDX).
	</Card>
	<Card title="Barre latérale & métadonnées" icon="setting">
		Ajustez `astro.config.mjs` (sidebar, titre, lien GitHub, `site` / `base`).
	</Card>
	<Card title="Prévisualiser en local" icon="laptop">
		`npm install` puis `npm run dev` — serveur sur http://localhost:4321
	</Card>
	<Card title="Médias" icon="document">
		Images dans `src/assets/` ou fichiers statiques dans `public/`.
	</Card>
