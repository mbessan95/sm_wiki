Synchro Run PCI

2× par sprint Dev × Run

La Synchro Run PCI est un rituel à **double sens** qui réunit l'équipe de développement et l'équipe de Run. C'est un moment d'échange autour des problématiques opérationnelles du moment.

-   Définition d'**actions concrètes** pour aider à réduire le run ou rendre l'équipe de run plus autonome
-   Les inputs de l'équipe de run permettent à l'équipe de **prioriser des sujets Lean** pour réduire le run futur
-   Alignement sur les **problématiques opérationnelles** du moment
